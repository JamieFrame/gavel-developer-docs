# gavel-developer-docs

Public builder documentation for The Gavel Protocol.

This repository is a placeholder. Builder documentation will be populated as
v2 nears mainnet launch. The audience is external developers building custom
credit products, custom orchestrators, custom curation layers, custom
frontends, and composition products on top of the v2 substrate.

For internal protocol-team documents, see the private gavel-protocol-docs
repository.
